# Form Validation

This is a simple contact form validation project built using HTML, CSS, and JavaScript.

## Features

- Validates full name with minimum character length
- Validates phone number as a 10-digit number
- Validates email format
- Validates message with minimum length
- Shows instant validation messages while typing
- Prevents form submission if any field is incorrect
- Shows success alert when all fields are valid

## Technologies Used

- HTML
- CSS
- JavaScript

## How It Works

The form contains fields for full name, phone number, email, and message. JavaScript validation functions check each field when the user types. If the input is invalid, an error message is shown. If the input is valid, a check mark is displayed.

When the submit button is clicked, all fields are checked again. The form submits only if every field is valid.
